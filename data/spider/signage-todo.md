# Signage to-do — corridors that want billboards

As the spider builds legs, note here which corridors have real roadside
signage/attractions worth a billboard pass (Opus drafts, owner workshops; the
billboard copy ships as code PRs on the 1.9 line, never in map PRs). Billboards
need BUILT legs, so this list trails the map build.

## I-95 Southeast spine (built 2026-07-08)

- **South of the Border** — Dillon, SC, on `florence_sc_us -> lumberton_nc_us`
  (the SC/NC line). The definitive I-95 attraction: hundreds of miles of
  countdown billboards ("You never sausage a place!"), Pedro, the sombrero
  tower, fireworks, a huge diesel stop. A whole billboard *series* on its own —
  the parody register is basically pre-written by the real thing. Prime.
- **Pedro's countdown billboards** — the miles-out teasers are the exact
  short-hook `spoken:` shape; a natural multi-sign run down the SC stretch.
- **Walterboro, SC** ("front porch of the Lowcountry") + the Point South /
  Santee truck-stop clusters — lighter, but real I-95 roadside character.
- **Pending build:** the northern I-95 legs (Richmond -> DC -> Baltimore ->
  Philadelphia -> NJ -> NYC -> New England) are the next corridors; signage
  once they exist (South of the Border's northbound countdown, Maryland House,
  Delaware toll, etc.).

## Owner checkpoint requests (place when the road gets built)

- **Midlothian, VA** on **US-60** (Midlothian Turnpike) west/southwest of
  Richmond -- owner used to live there. No US-60 leg exists yet, so there is
  nowhere to place it now. It belongs on a future Richmond -> Powhatan ->
  Farmville -> Lynchburg US-60 corridor (US-60 "goes a ways"). Add Midlothian
  as an early checkpoint out of Richmond the moment that leg is built.

## Corridors unlocked (build next, then signage)

- **I-85 spine** — Petersburg, VA is now a node (I-95 x I-85 split). I-85 runs
  Petersburg -> Durham -> Greensboro -> Charlotte -> Atlanta and would connect
  to the EXISTING `charlotte -> greensboro` I-85 leg. High-value freight
  corridor, natural next build off this batch.
